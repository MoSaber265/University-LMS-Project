## 🚀 نظرة عامة عن المشروع
نظام ويب متكامل مصمم لتسهيل العملية الأكاديمية للطالب والإدارة. يتيح النظام للطالب تسجيل مواده، متابعة درجاته، ونسبة غيابه، بينما يوفر للإدارة لوحة تحكم كاملة لإدارة المحتوى التعليمي.

## ✨ المميزات الرئيسية (Features)
- **Student Dashboard:** عرض شامل لبيانات الطالب والـ GPA.
- **Academic Years System:** تقسيم المواد على 4 مستويات دراسية.
- **Course Registration:** نظام تسجيل مواد ذكي (Add to Cart logic).
- **Admin Panel:** لوحة تحكم كاملة لإضافة وحذف وتعديل البيانات.
- **Responsive Design:** واجهة مستخدم مريحة تعمل على جميع الأجهزة.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla).
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB Atlas (Cloud).
- **Version Control:** Git & GitHub.
